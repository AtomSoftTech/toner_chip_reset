
Connections
===========

Depending on the board the i2c pins are:

	Board           I2C / TWI pins
	Uno, Ethernet   A4 (SDA), A5 (SCL)
	Mega2560        20 (SDA), 21 (SCL)
	Leonardo        2 (SDA), 3 (SCL)
	Due             20 (SDA), 21 (SCL), SDA1, SCL1
